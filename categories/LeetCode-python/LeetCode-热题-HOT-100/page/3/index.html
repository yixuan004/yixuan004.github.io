йўҳзӣ®еӨ§ж„Ҹпјҡ

иҝҗз”ЁдҪ жүҖжҺҢжҸЎзҡ„ж•°жҚ®з»“жһ„пјҢи®ҫи®Ўе’Ңе®һзҺ°дёҖдёӘВ  LRU (жңҖиҝ‘жңҖе°‘дҪҝз”Ё) зј“еӯҳжңәеҲ¶ гҖӮ
е®һзҺ° LRUCache зұ»пјҡ

  • LRUCache(int capacity) д»ҘжӯЈж•ҙж•°дҪңдёәе®№йҮҸВ capacity еҲқе§ӢеҢ– LRU зј“еӯҳ
  • int get(int key) еҰӮжһңе…ій”®еӯ— key еӯҳеңЁдәҺзј“еӯҳдёӯпјҢеҲҷиҝ”еӣһе…ій”®еӯ—зҡ„еҖјпјҢеҗҰеҲҷиҝ”еӣһ -1 гҖӮ
  • void put(int key, int value)В еҰӮжһңе…ій”®еӯ—е·Із»ҸеӯҳеңЁпјҢеҲҷеҸҳжӣҙе…¶ж•°жҚ®еҖјпјӣеҰӮжһңе…ій”®еӯ—дёҚеӯҳеңЁпјҢеҲҷжҸ’е…ҘиҜҘз»„гҖҢе…ій”®еӯ—-еҖјгҖҚгҖӮеҪ“зј“еӯҳе®№йҮҸиҫҫеҲ°дёҠйҷҗж—¶пјҢе®ғеә”иҜҘеңЁеҶҷе…Ҙж–°ж•°жҚ®д№ӢеүҚеҲ йҷӨжңҖд№…жңӘдҪҝз”Ёзҡ„ж•°жҚ®еҖјпјҢд»ҺиҖҢдёәж–°зҡ„ж•°жҚ®еҖјз•ҷеҮәз©әй—ҙгҖӮ
