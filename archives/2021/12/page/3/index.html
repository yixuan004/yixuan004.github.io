йўҳзӣ®еӨ§ж„Ҹпјҡз»ҷдҪ дёҖдёӘ m x n зҡ„дәҢе…ғзҹ©йҳө matrix пјҢдё”жүҖжңүеҖјиў«еҲқе§ӢеҢ–дёә 0 гҖӮиҜ·дҪ и®ҫи®ЎдёҖдёӘз®—жі•пјҢйҡҸжңәйҖүеҸ–дёҖдёӘж»Ўи¶іВ matrix[i][j] == 0 зҡ„дёӢж ҮВ (i, j) пјҢе№¶е°Ҷе®ғзҡ„еҖјеҸҳдёә 1 гҖӮжүҖжңүж»Ўи¶і matrix[i][j] == 0 зҡ„дёӢж Ү (i, j) иў«йҖүеҸ–зҡ„жҰӮзҺҮеә”еҪ“еқҮзӯүгҖӮ

е°ҪйҮҸжңҖе°‘и°ғз”ЁеҶ…зҪ®зҡ„йҡҸжңәеҮҪж•°пјҢе№¶дё”дјҳеҢ–ж—¶й—ҙе’Ңз©әй—ҙеӨҚжқӮеәҰгҖӮ
